Search API Auto Suggestion
---------------------

This module extends Search API [1], Search API Solr [2], and Search API Spellcheck [3] to allow search services which can generate
spelling suggestions and provide users a google like search suggestion experience. Also, provide a common way for them to be
altered and themed.

[1] http://drupal.org/project/search_api
[2] http://drupal.org/project/search_api_solr
[3] http://drupal.org/project/search_api_spellcheck
